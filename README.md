# Customer Segmentation Project

## 📌 Objective
Segment customers based on their behavior using clustering techniques.

## 📊 Dataset
This dataset contains customer information such as age, income, and spending score.

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

## 🔍 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- K-Means Clustering

## 📈 Results
Identified different customer segments based on purchasing behavior.

## 🚀 Conclusion
This project helps businesses understand customer groups for targeted marketing.
